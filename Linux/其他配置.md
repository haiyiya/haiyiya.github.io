# 配置 ulimit

用户可打开最大文件数和线程数

```sh
sudo vi /etc/security/limits.conf
在文件末尾添加如下配置：(* 代表所有用户)
* soft nproc 10240
* hard nproc 10240
* soft nofile 65535
* hard nofile 65535
重新登录 ulimit -a 确认
```
