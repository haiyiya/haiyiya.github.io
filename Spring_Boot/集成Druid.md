# 监控页面

```yml
spring:
  datasource:
    druid:
      stat-view-servlet:
        enabled: true
        url-pattern: /druid/*
        login-username: test
        login-password: test
        # allow: 127.0.0.1
        # deny:
```

访问 http://127.0.0.1:8097/druid/login.html

> 经测试，allow不配置默认只允许127.0.0.1，配置后只允许配置的ip
